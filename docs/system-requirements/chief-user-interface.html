Software Requrements

Software Requirement IDSystem Requirement IDRequirement TitleRequirement Description
req_chf_ui_1_v1req_manual_action_1_v1Activate Run Duty CycleThe system should allow the activation of a single Duty Cycle for selected Nomadd's, primarily for internal purposes.
req_chf_ui_2_v1req_manual_action_2_v1Execute Find Home NowThe system should enable Nomadd(s) to find their docking position immediately upon request.
req_chf_ui_3_v1req_manual_action_3_v1Trigger Stop NowThe system should provide functionality to stop selected Nomadd(s), ensuring they cease current operations and return to an idle state.
req_chf_ui_4_v1req_manual_action_4_v1Reset Packet CountersThe system should offer a way to reset the error packet counters, ensuring accurate tracking of subsequent errors.
req_chf_ui_5_v1req_manual_action_5_v1Grant Run ClearanceThe system should allow manual granting of run clearance to specific nodes, providing control over operational permissions.
req_chf_ui_6_v1req_manual_action_6_v1Set Nomadd NameThe system should permit the setting or changing of a Nomadd's nickname, restricted to cases when only one Nomadd is selected.
req_chf_ui_7_v1req_manual_action_7_v1Record User CommentsThe system should provide a feature for users to add or modify comments related to the Nomadd system.
req_chf_ui_8_v1req_manual_action_8_v1Record Last Intervention DateThe system should allow technicians to set a date indicating when the Nomadd requires intervention, along with a comment section for detailing the reason.
req_chf_ui_9_v1req_manual_action_9_v1Display LoRa IDThe system should display the wireless ID of the Nomadd for identification and configuration purposes.
req_chf_ui_10_v1req_manual_action_10_v1Set Array NameThe system should allow users to view or set the name or code of the array where the Nomadd is located.
req_chf_ui_11_v1req_manual_action_11_v1Define Array LengthThe system should enable users to define or view the length of the array that the Nomadd is assigned to clean.
req_chf_ui_12_v1req_manual_action_12_v1Configure Finding Home TimeoutThe system should provide an option to configure the time allowed for a Nomadd to find its home or docking position.
req_chf_ui_13_v1req_manual_action_13_v1Activate Enable LockdownThe system should provide an option to activate lockdown mode, causing the Nomadd to return to the dock (if required) and apply maximum braking, suspending any scheduled runs.
req_chf_ui_14_v1req_manual_action_14_v1Deactivate Disable LockdownThe system should allow deactivation of lockdown mode, enabling the Nomadd to resume normal operations and scheduled runs.
req_chf_ui_15_v1req_manual_action_15_v1Adjust Logging LevelThe system should allow adjustment of the level of detail for logs generated by Nomadd.
req_chf_ui_16_v1req_manual_action_16_v1Toggle Maximum Clean ModeThe system should offer a toggle for 'Maximum Clean' mode, which when activated, disables any existing schedules and sets 'Cycles Per Event' to 1.
req_chf_ui_17_v1req_manual_action_17_v1Set Maximum Clean Start TimeThe system should allow setting of the start time for 'Maximum Cleaning Effort' period, in accordance with the site's timezone.
req_chf_ui_18_v1req_manual_action_18_v1Set Maximum Clean End TimeThe system should permit setting of the end time for 'Maximum Cleaning Effort' period, in line with the site's timezone.
req_chf_ui_19_v1req_manual_action_19_v1Set Max Cleaning VoltageThe system should enable configuration of the minimum voltage required to initiate a cleaning cycle in 'Maximum Cleaning Mode'.
req_chf_ui_20_v1req_manual_action_20_v1Toggle Schedule Active StateThe system should provide an option to toggle the active state of the predefined Cleaning Event Schedule.
req_chf_ui_21_v1req_manual_action_21_v1Set Time of Day for Run 1The system should allow setting the specific time for Cleaning Event run 1, expressed in minutes since local midnight.
req_chf_ui_22_v1req_manual_action_22_v1Set Time of Day for Run 2The system should allow setting the specific time for Cleaning Event run 2, measured in minutes since local midnight.
req_chf_ui_23_v1req_manual_action_23_v1Set Number of Cycles Per EventThe system should provide a feature to set the number of Duty Cycles per Cleaning Event that a Nomadd will perform.
req_chf_ui_24_v1req_manual_action_24_v1Set Brush Spin-up DurationThe system should enable configuration of the duration for the Brush motor's spin-up time before the Rail Motor is activated during a Duty Cycle.
req_chf_ui_25_v1req_manual_action_25_v1Set Duty Cycle Rail Motor SpeedThe system should allow setting of the speed percentage of the motor that moves the Nomadd during a Duty Cycle.
req_chf_ui_26_v1req_manual_action_26_v1Set Duty Cycle Brush Motor SpeedThe system should allow configuration of the speed percentage of the motor that turns the brush on the Nomadd during a Duty Cycle.
req_chf_ui_27_v1req_manual_action_27_v1Set High Speed LimitThe system should provide an option to set the upper speed limit of the rail motor during a Duty Cycle, preventing the motor from stopping too fast due to high wind.
req_chf_ui_28_v1req_manual_action_28_v1Configure Leave Dock TimeoutThe system should allow setting the duration the Nomadd waits to disengage from the dock before an alert is raised and survival mode is entered.
req_chf_ui_29_v1req_manual_action_29_v1Set Forwards Find Panel TimeoutThe system should provide an option to set the timeout for the Nomadd to leave the dock and reach the panels in a forward direction during a Duty Cycle.
req_chf_ui_30_v1req_manual_action_30_v1Set Forwards Panel Clean TimeoutThe system should allow setting the timeout for the Nomadd to clean the panel in a forward direction during a Duty Cycle.
req_chf_ui_31_v1req_manual_action_31_v1Set Forwards Clean Stop DelayThe system should offer a feature to set the delay time after the Nomadd has left the panel going forward before stopping to ensure complete cleaning.
req_chf_ui_32_v1req_manual_action_32_v1Set Straighten Up DurationThe system should allow configuration of the duration the Nomadd will stop and spin the brush backward to straighten up at the pass of the panels before reversing to clean backward.
req_chf_ui_33_v1req_manual_action_33_v1Set Dock Straighten Up DurationThe system should enable setting the duration the Nomadd will spin the brush backward to straighten up when in the dock.
req_chf_ui_34_v1req_manual_action_34_v1Set Backwards Panel TimeoutThe system should provide an option to set the timeout for the Nomadd to reach the panels in a backward direction after turning around during a Duty Cycle.
req_chf_ui_35_v1req_manual_action_35_v1Set Backwards Panel Clean TimeoutThe system should allow setting the timeout for the Nomadd to clean the panel in a backward direction during a Duty Cycle.
req_chf_ui_36_v1req_manual_action_36_v1Set Docking SpeedThe system should allow configuration of the speed percentage of the rail motor during docking.
req_chf_ui_37_v1req_manual_action_37_v1Set Docking TimeoutThe system should offer a feature to set the expected time for the unit to find the docking position after finishing cleaning the panels backward.
req_chf_ui_38_v1req_manual_action_38_v1Set Motor1O Over Current LimitThe system should allow setting of the over-current limit for Motor1O.
req_chf_ui_39_v1req_manual_action_39_v1Set Motor2O Over Current LimitThe system should allow setting of the over-current limit for Motor2O.
req_chf_ui_40_v1req_manual_action_40_v1Set Motor3O Over Current LimitThe system should allow setting of the over-current limit for Motor3O.
req_chf_ui_41_v1req_manual_action_41_v1Set Motor4O Over Current LimitThe system should allow setting of the over-current limit for Motor4O.
req_chf_ui_42_v1req_manual_action_42_v1Set Motor5O Over Current LimitThe system should allow setting of the over-current limit for Motor5O.
req_chf_ui_43_v1req_manual_action_43_v1Set Over Current Limit DurationThe system should provide a feature to set the duration for the over-current limit for both the Brush and Rail Motor.
req_chf_ui_44_v1req_manual_action_44_v1Set Supply Charge needed to runThe system should enable configuration of the battery voltage level required to commence a Duty Cycle.
req_chf_ui_45_v1req_manual_action_45_v1Set Supply Charge WarningThe system should allow setting the voltage level at which an alert for low supply voltage is generated.
req_chf_ui_46_v1req_manual_action_46_v1Set Supply Charge CriticalThe system should offer a feature to set the critical supply voltage level at which an alert is sent and survival mode is entered if the unit is not docked.
req_chf_ui_47_v1req_manual_action_47_v1Set Internal Temp HighThe system should allow setting the internal temperature limit that causes all activity to halt and survival mode to be entered.
req_chf_ui_48_v1req_manual_action_48_v1Set Internal Temp CriticalThe system should enable setting the critical internal temperature at which all activity halts and the system enters a cooling phase.
req_chf_ui_49_v1req_manual_action_49_v1Configure Docking DelayThe system should provide an option to set the docking delay between docking and stopping the motors, useful for dock locking systems.
req_chf_ui_50_v1req_manual_action_50_v1Set Minutes Until OfflineThe system should allow setting the number of minutes until the node considers the chief offline.
req_chf_ui_51_v1req_manual_action_51_v1Activate Enable Debug ModeThe system should offer an option to activate the debug mode for troubleshooting and development purposes.
req_chf_ui_52_v1req_manual_action_52_v1Deactivate Disable Debug ModeThe system should allow deactivation of the debug mode to resume normal operational modes.
req_chf_ui_53_v1req_manual_action_53_v1Execute Commit Debug Mode ChangesThe system should enable committing of the current debug mode changes to flash storage.
req_chf_ui_54_v1req_manual_action_54_v1Activate Enable Burn-in ScheduleThe system should provide an option to activate the burn-in schedule for intensive testing and calibration.
req_chf_ui_55_v1req_manual_action_55_v1Deactivate Disable Burn-in ScheduleThe system should allow deactivation of the burn-in schedule to resume standard operational schedules.
req_chf_ui_56_v1req_manual_action_56_v1Set LoRa Power LevelThe system should enable setting of the LoRa transmission power level to control the range and signal strength.

req_chf_ui_58_v1req_manual_action_58_v1Toggle Node Auto Find HomeThe system should provide an option to toggle the automatic home-finding feature of the Nomadd when it is undocked and idle, offering control over the node's automatic behaviors.
req_chf_ui_59_v1req_manual_action_59_v1Set Finding Home SpeedThe system should allow setting of the speed at which the Nomadd attempts to find its home, providing control over the efficiency and speed of the home-finding process.
req_chf_ui_60_v1req_manual_action_60_v1Set Max Rail Motor RPMThe system should provide a feature to set the maximum speed of the rail motor in RPM, offering control over the operational speed of the Nomadd.
req_chf_ui_61_v1req_manual_action_61_v1Set Max Brush Motor RPMThe system should allow configuration of the maximum speed of the brush motor in RPM, offering precise control over the brushing operations.
req_chf_ui_62_v1req_manual_action_62_v1Set LoRa ChannelThe system should provide an option to set the LoRa channel used by the Nomadd, ensuring proper communication and signal strength.

req_chf_ui_65_v1req_manual_action_65_v1Set LoRa Data Speed RateThe system should provide an option to set the LoRa data speed rate, offering control over the data transmission speed and range.
req_chf_ui_66_v1req_manual_action_66_v1Toggle Docking: Clean BrushThe system should offer a toggle to enable or disable the brush cleaning feature in the dock, ensuring proper maintenance and cleanliness of the brush.
