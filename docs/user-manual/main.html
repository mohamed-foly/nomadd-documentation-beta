Features​

Sidebar​

Actions​

Run Duty Cycle​

This is only internal now. We use cleaning event instead... Instruct the Selected Nomadd's to run a single Duty Cycle now.

Find Home Now​

Instruct Nomadd(s) to find docking position now.

Stop Now​

Stop selected Nomadd(s). This is a good way to have a nomadd stop what it's doing and return to the idle state.

Reset Packet Counters​

Reset the error packet counter.

General​

Give Run Clearance​

Manually give a node run clearance.

Nomadd Name​

Nomadd Nickname. Can only be changed when one nomadd selected.

Comment​

User Specified Comment.

Last Intervention Date​

For Technicians only. Set this whenever the Nomadd needs Intervention of any kind. Use Comment to record reason.

LoRa ID​

Wireless ID of this Nomadd.

Array Name​

Name or code of the array the Nomadd resides upon.

Array Length​

The length of the array this Nomadd cleans.

Finding Home Timeout​

The amount of time given for the node to successfully find home.

Enable Lockdown​

Return Nomadd to the dock (If Needed) and apply maximum braking. No scheduled runs will be performed.

Disable Lockdown​

Disable lockdown so a Nomadd can run normally.

Logging Level​

Level of detail of Nomadd Logs to make.

Schedule​

Maximum Clean?​

Will this Nomadd maximise its cleaning efforts? Note: Switching this on will disable any existing schedule(s) and set Cycles Per Event to 1.

Maximum Clean Start Time​

Start of Maximum Cleaning Effort Period. (site timezone applies)

Maximum Clean End Time​

End of Maximum Cleaning Effort Period. (site timezone applies)

Max Cleaning Voltage Required​

Minimum Voltage required to run a Cycle when in Maximum Cleaning Mode.

Schedule Active?​

Is the defined Cleaning Event Schedule active?

Time of Day Run 1​

Number of minutes since local midnight to begin Cleaning Event run 1 on any day.

Time of Day Run 2​

Number of minutes since local midnight to begin Cleaning Event run 2 on any day.

Run Settings​

Number of Cycles Per Event​

Number of Duty Cycles per Cleaning Event this Nomadd will perform.

Brush Spin-up Duration​

After Starting the Brush motor, a pause of this duration (in Seconds) is executed prior to turning the Rail Motor on.

Duty Cycle Rail Motor Speed​

The speed ( % of Max) of the motor that moves the Nomadd during a Duty Cycle.

Duty Cycle Brush Motor Speed​

The speed ( % of Max) of the motor that turns the brush on the Nomadd during a Duty Cycle.

High Speed Limit​

The speed (% of Max) of the rail motor that is the upper limit before the motor will be stopped for being too fast. This prevents wind from blowing the Nomadd around.

Leave Dock Timeout​

When the Nomadd begins its cycle, it will wait for this duration in seconds to disengage from the dock before an alert is raised and survival mode entered.

Forwards Find Panel Timeout​

When beginning a Duty Cycle, the Nomadd will raise an alert and enter survival mode if it cannot leave the dock and cross onto the panels within this amount of time (Seconds).

Forwards Panel Clean Timeout​

When beginning a Duty Cycle, the Nomadd will raise an alert and enter survival mode if it cannot clean the panel in the forwards direction in this amount of time (Seconds).

Forwards Clean Stop Delay​

Once the Nomadd has left the panel going forward, it will continue for this many deciseconds (one-tenth of a second) before stopping to ensure complete cleaning.

Straighten Up Duration​

The duration in seconds (0=Disabled) the Nomadd will stop and spin the brush backward to straighten up at the one pass of the panels before reversing to clean backward.

Dock Straighten Up Duration​

The duration in seconds (0=Disabled) the Nomadd will spin the brush backward to straighten up when in the dock.

Backwards Panel Timeout​

After turning around, the Nomadd will raise an alert and enter survival mode if it does not cross onto the panels within this amount of time (in seconds).

Backwards Panel Clean Timeout​

When beginning a Duty Cycle, the Nomadd will raise an alert and enter survival mode if it cannot clean the panel in the backwards direction in this amount of time (Seconds).

Docking Speed​

Speed in % of the rail motor during docking.

Docking Timeout​

After finishing cleaning the panels backwards, the unit expects to find the docking position within this time.

Set Points​

Motor1O Over Current Limit​

Motor1O Over Current Limit.

Motor2O Over Current Limit​

Motor2O Over Current Limit.

Motor3O Over Current Limit​

Motor3O Over Current Limit.

Motor4O Over Current Limit​

Motor4O Over Current Limit.

Motor5O Over Current Limit​

Motor5O Over Current Limit.

Over Current Limit Duration​

Over Current Limit Duration for Both the Brush and Rail Motor.

Supply Charge needed to run​

A Duty Cycle will not commence if battery voltage is below this level.

Supply Charge Warning​

Specifies the point at which an alert will be generated for Low Supply Voltage.

Supply Charge Critical​

When Supply voltage falls below this level, Alert will be send and survival mode entered if the unit is not docked.

Internal Temp High​

(Degrees C) The Internal Temp which causes all activity to halt and survival mode to be entered.

Internal Temp Critical​

(Degrees C) The Internal Temp which causes all activity to halt and Wait For Cool Behaviour to be invoked.

Docking Delay​

Delay in 100ms between docking and stopping the motors. For use with the dock locking system.

Minutes Until Offline​

Number of minutes until the node should consider chief offline.

Debug​

Enable Debug Mode​

Enable Debug Mode.

Disable Debug Mode​

Disable Debug Mode.

Commit Debug Mode Changes​

Commit debug current mode changes to flash.

Enable Burn-in Schedule​

Enable Burn-in Schedule.

Disable Burn-in Schedule​

Disable Burn-in Schedule.

Setup​

LoRa Power Level​

The LoRa transmission power level.

Alerts Enabled?​

Does this Nomadd generate alerts and alert emails? Switch this off during maintenance to prevent alerts being generated.

Node Auto Find Home?​

Will this Nomadd automatically find home when it is undocked and idle? This is a node function.

Finding Home Speed​

Speed of the Nomadd (% of Max RPM) when attempting to Find Home.

Max Rail Motor RPM​

Maximum Rail Motor Speed in RPM (1-150).

Max Brush Motor RPM​

Maximum Brush Motor Speed in RPM (1-150).

LoRa Channel​

The LoRa Channel used for this Nomadd. This change does not become effective until the node is power cycled.

Re-Initilise​

Re-apply All known parameter values to this Nomadd.

Temperature Calibration​

The internal temperature calibration. (1 unit = 0.5C).

LoRa Data Speed Rate​

The LoRa Data Speed Rate. Lower values = More range and reduced data throughput. Power cycle the Nomadd after setting this for changes to take effect.

Docking: Clean Brush?​

Will the Nomadd clean its brush in the dock?

Docking Brush Clean Time​

Time to pause Docking while the brush is cleaning during Docking.

Serial Debug​

Log Serial Traffic. For developers only. DO NOT LEAVE THIS ON FOR LONG PERIODS!

Set Defaults​

Sets the defaults for initialization of all new Nomadds based on the current selection.

Delete Nomadd​

Remove Nomadd from the Database. Nomadd can be re-added manually or power cycle the Nomadd to automatically re-add it.

Pair Mode?​

Is this node operating in pair mode?

Exclusion Group ID​

Exclusion group ID, shared by pairs or groups of nomadds that should not run at the same time.

Run Clearance Timeout​

Number of seconds until the node stops waiting for clearance on a run and terminates the run.

Modbus Unit ID​

The TCP Modbus Unit ID this Nomadd will have when exposed to Modbus.

Delete TCP Modbus Unit ID​

Delete the TCP Modbus Unit ID from this unit.

Motor Directions​

The initial orientation of the motors coming out of the dock.

Rail Motor Commutation Multiplier​

The conversion between the number of commutation cycles per 10th of a second and RPM for the rail motor, e.g., for a gearbox ratio of 28, with a 4 pole 3 phase motor, it would be 10 (28 4 * 3) / 60 = 56.

Brush Motor Commutation Multiplier​

The conversion between the number of commutation cycles per 10th of a second and RPM for the brush motor, e.g., for a gearbox ratio of 28, with a 4 pole 3 phase motor, it would be 10 (28 4 * 3) / 60 = 56.

Search​

Live search in all table's rows with the text value of name or array or status

Add​

Opens a popup to manually add a new node

Nodes Table​

Select​

select one nomadd device or more to apply action on it

Name​

Name the unit array considering the customer input.

Array​

Status​

View all statuses

Indicates current statues of node colored by configurable settings in configuration Status Colours
